Alətlər qatı

İnsanların hər gün istifadə edə biləcəyi sürətli və faydalı alətlər. Aydın kateqoriyalar, sadə giriş və böyüməyə uyğun struktur.

📚

Resurslar qatı

Seçilmiş saytlar, bələdçilər, izahlar və kolleksiyalar sayəsində platforma axtarıla bilən bilik bazasına çevrilir.

💬

İcma qatı

Gələcəkdə istifadəçilərin sual verə, cavab ala, təcrübə paylaşa və etibar yarada biləcəyi müzakirə sistemi.

Onu fərqli edən nədir?

Brend əsaslı struktur: OrikLab, daha böyük OrikOS vizyonu altında ayrıca məhsul kimi işləyir.
Modul inkişaf: kiçik başla, ağıllı böyü və memarlıq üzərində tam nəzarəti saxla.
Məxfilik yönümlü təməl: hüquqi səhifələr və texniki baza artıq qurulub.
Beynəlxalq istiqamət: çoxdilli və qlobal auditoriya üçün genişlənməyə hazırdır.

Hazırkı mərhələ

Bu versiya peşəkar başlanğıcdır: xüsusi domain, canlı deploy sistemi, hüquqi səhifələr və ilk brend interfeysi artıq hazırdır.

Domain qoşulub: oriklab.com artıq Vercel üzərində aktivdir.
GitHub workflow: yeniləmələr versiya nəzarəti ilə peşəkar şəkildə idarə olunur.
Hüquqi baza: Məxfilik və Impressum səhifələri artıq mövcuddur.
Növbəti addım: bunu interaktiv və daha güclü platformaya çevirməkdir.

Bu, OrikLab-ın yalnız başlanğıcıdır.

Hazırkı səhifə peşəkar təməldir. Növbəti mərhələdə layihə xüsusi dizayn, kateqoriya səhifələri, real alətlər və genişlənə bilən icma sistemi ilə tam məhsula çevrilə bilər.
